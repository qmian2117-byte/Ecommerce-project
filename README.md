A full-stack **Ecommerce Website** built with **HTML, CSS (Tailwind), JavaScript (Frontend)** and **Node.js + Express + MongoDB (Backend)**.  
This project features a responsive design, dynamic product pages, user authentication, a working cart system, and email OTP verification.

## ğŸ§© Features

### Frontend
- Responsive design for desktop, tablet, and mobile.
- Dynamic product listing and product detail pages.
- Search bar with live filtering.
- Login/Signup forms with OTP verification.
- Cart functionality with add/remove items.
- Mobile-friendly navigation menu with toggle.

### Backend
- Node.js + Express server for API endpoints.
- MongoDB database for storing users, products, and orders.
- User authentication using JWT.
- CRUD operations for products and orders.
- Email notifications via Nodemailer.
